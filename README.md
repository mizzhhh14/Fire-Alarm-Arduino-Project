# Fire-Alarm-Arduino-Project
🔥 Fire Alarm using Arduino and Tinker Cad

## 📌 Project overview
I built a simple **Fire Alarm System** that detects possible fire conditions using **Temperature sensor** and **Gas sensor**. When the sensor value crosses a certain value, the system activates a **Buzzer** and led alert.

## 🛠️ Components used
- Arduino UNO
- TMp36 Temperature sensor
- MQ-2 Gas sensor
- Buzzer
- LEDs
- Resistors
- Breadboard
- Jumper wires

## ⚡How it works
The Arduino constantly monitors-
-  🌡️ temperature levels
-  💨 gas/smoke levels

If a dangerous situation is detected
- 🚨 Buzzer turns ON
- 🔴 LED lights up
